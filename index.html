üéØ NOLA Financial Intelligence Platform

Problem: CFOs spent 2-4 hours per quarter manually analyzing financial data across 38 charter schools with no standardized health scoring or predictive capabilities.

Solution: Built comprehensive Streamlit platform with automated health scoring (0-100), ML forecasting, and executive dashboards reducing analysis time by 95%.

Key Features:

  • Executive Summary with portfolio-wide health cards
  • FinancialHealthAnalyzer class (weighted CSAF scoring)
  • 10+ ML models (XGBoost, HGBR, Random Forest, Neural MLP)
  • Automated insight generation engine
  • Bootstrap confidence intervals for predictions
  • Interactive Plotly visualizations
